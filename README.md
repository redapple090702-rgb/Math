# ✈️ 삼각함수와 등차수열을 이용한 시차 적응 동적 시뮬레이션

본 프로젝트는 고등학교 **대수(Algebra)** 교과 과정에서 학습한 **삼각함수의 주기·평행이동 성질**과 **수열의 귀납적 정의(등차수열 점화식)**를 수리생물학적 현상에 융합한 탐구입니다. 

한국에서 미국 뉴욕으로 이동 시 발생하는 일주기 생체 리듬(Circadian Rhythm)의 파형 충돌과 시차 적응 과정을 수리적으로 모델링하고, 날짜별 위상 변화를 파이썬(Python) 애니메이션으로 실시간 시각화했습니다.

---

## 🧮 수학적 배경 (Mathematical Background)

### 1. 현지 표준 일주기 리듬 함수
뉴욕 현지 표준 생체 각성도 곡선입니다. 오후 4시(16시)에 최고 컨디션 10점, 새벽 4시에 최저점 0점을 갖도록 평행이동을 적용했습니다.
* f(t) = 5 * sin( (pi / 12) * (t - 10) ) + 5

### 2. 시차 적응에 따른 신체 기능 함수 (n일차)
귀납적 점화식에 따라 매일 1시간씩 생체 시계가 뒤로 미뤄지는 위상 지연(Phase Delay)을 반영한 등차수열 모델입니다.
* g(t) = 5 * sin( (pi / 12) * (t - a_n) ) + 5
* 초기 조건: a_1 = 21 (뉴욕 도착 첫날, 현지 시간 기준 오전 3시에 내 신체 정점이 오는 시차 충돌 현상)
* 귀납적 점화식: a_n+1 = a_n - 1.0 (단, 목표 위상값인 a_n이 10이 될 때까지 매일 공차 -1씩 적용)

### 3. 시차 고통 지수 (JPI, Jet Lag Pain Index)
현지 표준 리듬 f(t)와 내 신체 리듬 함수 g(t)의 격차를 차함수의 절댓값 평균으로 정의하여 피로도를 정량화했습니다.
* JPI = 평균( |f(t) - g(t)| )

## 💻 실시간 대시보드 시뮬레이션 기능

코드를 실행하면 DAY 1부터 DAY 12까지 시간이 흐름에 따라 다음과 같은 동적 연산이 실시간으로 수행됩니다.

1. **터미널 실시간 수치 출력:** 매일 변화하는 신체 최고점 시간대와 시차 고통 지수(JPI)가 실시간으로 계산되어 대시보드 형태로 출력됩니다.
2. **그래프 동적 애니메이션 (Real-time Phase Shift):** 빨간색 신체 리듬 곡선이 매일 1시간씩 현지 표준 시간축선(검은 점선)을 향해 스르륵 이동합니다.
3. **상태 변화 연산:** 등차수열 일반항 연산 결과(22 - n = 10)와 정확히 일치하는 **12일 차**가 되면, 그래프가 초록색으로 변하며 완벽한 동기화(시차 적응 완료)를 시각적으로 증명합니다.
