# ✈️ 삼각함수와 등차수열을 이용한 시차 적응 수학적 모델링

본 프로젝트는 고등학교 **대수(Algebra)** 교과 과정에서 학습한 **삼각함수의 주기·평행이동 성질**과 **수열의 귀납적 정의(등차수열 점화식)**를 수리생물학적 현상에 융합한 탐구입니다. 해외 이동 시 발생하는 일주기 생체 리듬(Circadian Rhythm)의 파형 충돌과 시차 적응 과정을 수리적으로 모델링하고 파이썬으로 시뮬레이션했습니다.

---

## 🧮 수학적 배경 (Mathematical Background)

### 1. 현지 표준 일주기 리듬 함수
뉴욕 현지 표준 생체 각성도 곡선입니다. 오후 4시(16시)에 최고 컨디션 $10$, 새벽 4시에 최저점 $0$을 갖도록 평행이동을 적용했습니다.
$$f(t) = 5 \sin\left(\frac{\pi}{12}(t - 10)\right) + 5$$

### 2. 시차 적응에 따른 신체 기능 함수 (n일차)
귀납적 점화식($a_1 = 21$, $a_{n+1} = a_n - 1.0$)에 따라 매일 1시간씩 생체 시계가 뒤로 밀리는 위상 지연(Phase Delay)을 반영한 등차수열 모델입니다.
$$g(t) = 5 \sin\left(\frac{\pi}{12}(t - a_n)\right) + 5$$
* **초기 조건:** $a_1 = 21$ (뉴욕 도착 첫날, 현지 시간 기준 오전 3시에 내 신체 정점이 오는 시차 충돌 현상)
* **귀납적 점화식:** $a_{n+1} = a_n - 1.0$ (단, 목표 위상값인 $a_n \ge 10$ 일 때까지 매일 공차 $-1$씩 적용)

### 3. 시차 고통 지수 (JPI, Jet Lag Pain Index)
현지 표준 리듬과 내 신체 리듬 함수의 격차를 **차함수의 절댓값**으로 정의하여 피로도를 정량화했습니다.
$$\text{JPI} = |f(t) - g(t)|$$
